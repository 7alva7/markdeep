Pilots use control panels, joysticks, or computers for precise movements like docking, changing orbits, or maintaining attitude. Reaction Control System (RCS) : Small thrusters (cold gas or liquid) positioned around the craft to provide precise rotations (pitch, roll, yaw) and small translations (up, down, left, right). Control Moment Gyroscopes (CMGs) : Spinning flywheels inside the spacecraft; changing their spin direction creates torques that rotate the entire vehicle without using fuel. Gimbaled Thrusters : Main engines can swivel (gimbal) to direct thrust, causing rotation. Magnetorquers : Devices that interact with Earth's magnetic field to provide torque, common for Earth-orbiting satellites. Orbital Maneuvers (Orbital Burns) : Firing main engines for significant changes in speed and orbit, like transferring to a new orbit or escaping gravity wells. Gravity Assist (Slingshot) : Using a planet's gravity to alter a spacecraft's speed and direction, saving fuel. ## Engineering ### Bussard Couplers ``` .------------------------------------. | | v _________|__________ .--+--. .---------. | | / \ | | | Sublight Matter | / Fuel \ | Impulse +------>| Recapture Scoop | \ Matrix / | Drive +------>|____________________| \ / | | |____________________| '--+--' '---------' |____________________| | ^ | | | v '------------' Filter Exhaust Port ``` ### Details A fast moving spacecraft scoops up hydrogen from the interstellar medium using an enormous funnel-shaped magnetic field (ranging from kilometers to many thousands of kilometers in diameter); the hydrogen is compressed until thermonuclear fusion occurs, which provides thrust to counter the drag created by the funnel and energy to power the magnetic field. ## Planetary Survey ### Deep Scans ### Geological Survey ### Xenobiology Review ### Anomalies
